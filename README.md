# 🚴‍♂️ VeloDash - Bike Sharing Data Analysis Dashboard

Ini adalah proyek *data analysis* untuk **Bike Sharing Dataset**. Proyek ini mencakup pemrosesan data, analisis eksploratif (EDA), dan penyajian data interaktif menggunakan *dashboard* Streamlit.

## 🛠️ Setup Environment
Berikut adalah langkah-langkah untuk menyiapkan *environment* sebelum menjalankan *dashboard*:

### Menggunakan Virtual Environment (Disarankan)
1. Buka Terminal / Command Prompt Anda.
2. Arahkan direktori aktif ke folder proyek ini (`submission`).
3. Buat *virtual environment* baru:
   ```bash
   python -m venv env
   ```
4. Aktifkan *virtual environment* tersebut:
   - **Windows:**
     ```bash
     env\Scripts\activate
     ```
   - **Mac/Linux:**
     ```bash
     source env/bin/activate
     ```
5. Instal semua *library* yang dibutuhkan:
   ```bash
   pip install -r requirements.txt
   ```

## 🚀 Cara Menjalankan Dashboard
Setelah semua *library* berhasil diinstal, Anda dapat langsung menjalankan *dashboard* Streamlit dengan perintah berikut di Terminal:

```bash
cd dashboard
streamlit run dashboard.py
```

Streamlit akan secara otomatis membuka *browser* Anda dan menampilkan *dashboard* pada alamat lokal (biasanya `http://localhost:8501` atau `http://localhost:8502`).
